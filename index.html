Roen's Website

This site. Built to introduce myself, show what I've been working on, and give people a real place to find my projects instead of a scattered GitHub profile.

html css js

I wanted something that looked and felt like the tools I actually use every day, a terminal, a dark editor theme, monospace type, rather than a generic template. Built it in plain HTML, CSS, and JS on purpose, no framework overhead for something this size.

Still adding to it as I build more. The playground page is where I'm testing small handwritten JS ideas outside of the main site.

2026 · NTU SummerBuild GitHub ↗

SummerBuild

A React Native app that helps NTU students find, host, and join pickup sports games on campus in seconds. Built with Expo, with a full light and dark theming system running through the whole app.

react native expo typescript

Started this because finding a pickup game on campus usually meant scrolling through a messy group chat. SummerBuild gives it a proper interface instead, browse open games, see who's in, join with one tap.

Most recently I built out a complete dark mode system, a ThemeContext with light and dark palettes wired through a useTheme() hook, applied across the whole codebase rather than bolted on after. Built as part of a team for NTU SummerBuild 2026.

projects

SummerBuild

A React Native app that helps NTU students find and join pickup sports games on campus. Built with Expo, with a full light and dark theming system running through the whole app.

react native expo typescript
source ↗

Started this because finding a pickup game on campus usually meant scrolling through a messy group chat. SummerBuild gives it a proper interface instead: browse open games, see who's in, join with one tap.

Most recently I built out a complete dark mode system, a ThemeContext with light and dark palettes wired through a useTheme() hook, applied inline across the whole codebase rather than bolted on after.

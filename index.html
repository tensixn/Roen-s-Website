Roen's Website

The site you are currently on. Built to introduce myself, show what I've been working on, and give people a real place to find my projects instead of a scattered GitHub profile.

html css js

I wanted something that looked and felt like the tools I actually use every day, a terminal, a dark editor theme, monospace type, rather than a generic template. Built it in plain HTML, CSS, and JS on purpose, no framework overhead for something this size.

2026 · NTU SummerBuild GitHub ↗

NTU Sports Matching App

A React Native app that helps NTU students find, host, and join pickup sports games on campus in seconds. Built with Expo, with a full light and dark theming system running through the whole app.

react native expo typescript

Built for a hackathon and started this because finding a pickup game on campus usually meant scrolling through a messy group chat. The matching app gives it a proper interface instead, browse open games, see who's in, join with one tap.

Most recently I built out a complete dark mode system, a ThemeContext with light and dark palettes wired through a useTheme() hook, applied across the whole codebase rather than bolted on after.

Page Summarizer

A Chrome extension that summarizes whatever webpage you're currently browsing using AI, built with a small team in a single hackathon sprint.

chrome extension javascript node

I worked on the extension side, the part that actually sits in your browser, pulls the page content, and shows the summary back to you. A teammate handled the Node server that talks to the AI model. We built this to help those with disabilities access information more easily.

2025 · SC2002 group project GitHub ↗

Turn-Based Combat

A turn-based combat game built in Java for my OOP module, built with a team of four applying SOLID principles and proper design patterns instead of just making it work.

java oop design patterns

I was the Items and Mechanics Expert on the team, responsible for the item system and how combat mechanics actually resolved turn to turn. The assignment cared as much about the architecture as the gameplay, so a lot of the real work was in keeping the class structure clean enough that four people could work on it without stepping on each other.

2026 · personal project GitHub ↗

Stoxed

A Telegram bot for tracking stocks and crypto, with price lookups, AI-powered predictions, watchlists, and portfolio tracking, right from a chat window. Search @StoxedBot on Telegram.

node telegraf claude api

Built this because checking prices meant switching apps constantly, a Telegram bot meant it lived right where I was already texting people anyway. It handles price lookups, watchlist management, portfolio tracking, and a P&L calendar, plus AI-powered predictions through Claude's API for a second opinion on a ticker.

Deployed on Railway so it's actually live and usable, not just something that runs on my laptop.
